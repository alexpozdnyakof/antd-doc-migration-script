# Ant Design Docs Migrator

Node app for bundle ant design mdx to storybook mdx stories. Builded using functional architecture a.k.a shell.

## Install

First install dependencies

```bash
$ yarn install
$ npm install
```

## Run

```bash
$ node main.mjs /target/path /result/path
```

## Tests

```bash
$ node --test --watch
```
